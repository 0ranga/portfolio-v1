Piano scales

Visual guide to better learn the scales for pianists

UX Designer

March 2019

About the project

One of the most fundamental parts of learning a musical instrument is to learn the scales. The basic ones are the major and minor scales which are the first ones learned by any musician, whether you follow a classical path or even a jazz path.

The challenge is to know which fingers are supposed to go on which key to be able to go up and down the keyboard by following the notes of the scale.

How the fingerings work on the piano

As a pianist myself, I have always found the process of learning the scales on the piano quite unadapted to the instrument. In fact, the piano is a very visual instrument and it can help a lot to learn it visually when it makes more sense than with a sheet.

Note that I have nothing against music theory and music notation, if anything I encourage anyone that wants to learn a musical instrument to get a good grasp of music theory and notation as it enables you to have the tools to understand music thoroughly.

However, in some cases like this one, there may be tools more useful.

Design challenges

There are 12 different notes (C, C#, D, D#, E, F, F#, G, G#, A, A#, B). For each of these notes there is a corresponding major scale and a corresponding minor scale (there are a lot of other types of scale but these two are the most used in western music).

This is what a typical exercise looks like:

Example of scale exercise from "Scales arpeggios and candences for piano" by Jeff Manookian

On this page we see only two scales, we have 24 of them when we add up the major and the minor scales. That would make 12 pages to display the same exercise as in the example above. Of course, on this page there are rythmic exercises that may be omitted and we could probably fit more scales on a page to use it as a reference tool. The rythmic exercises can be easily created by the musician.

One of my goal was to have some kind of reference tool that would be user-friendly and skimmed with ease.

Moreover, it is not just about the size of the document, as said before it is also about the way we learn the scales. A sheet cannot help that much at visualizing the notes of the scales on the keyboard. It cannot really help either at visualizing which scale is related to which or which ones are close.


In the end what I wanted to create was a reference tool that would be ergonomic, compact, useful and visual.

Design solutions

The final result is a one page document on which we can see all the major and minor scales along a circle of fifths so that we can see how they are related to one another.

All the information needed is here: the major scales, their relative minor scales, the circle of fifths, the key signatures and the fingerings.

You can download for free the full-sized pdf at archive.org/details/pianoscaleschart (there are versions with major scales only and minor scales only).

Piano scales reference tool in black and white

To get there I had to go through several different steps.

The circle of fifths is a wonderful tool used by a lot of musicians (not only pianists) to have a visual representation of all the scales relationships.

Example of circle of fifths

The example used above is a very good one as it includes the major scales but also their relative minor scales and their key signatures.

A little more about that:
• each major scale has a relative minor scale containing the same notes but starting on a different one
• the key signatures are at the beginning of a staff, this is what tells you which scale is used on a sheet

If you want to know more about all of that I invite you to look into music theory.

I chose to include all these elements in the product, here is the first stage:

First stage of the project

Once this was done, a few ideas came to my mind to make it better:
• make the black keys of the scales darker so that we can differentiate them from the white keys
• add the fingerings


Also, I posted it on reddit.com/r/piano to get users' input about it.

Along with minor corrections this is what came most from other users:
• the fingerings should be added
• something like a background should be added to make it more readable as some keyboards are not directly aligned with the scale names
• the outermost rim with the keys should contain the major scales instead of the minor ones to make it seamless with the central circles where the outermost rim contains the names of the major scales
• a lowercase "m" should be added next to the minor scale letters as it is how they are commonly spelled
• the key signatures should also contain the treble clef (for the left hand) just like it does in any piano sheet
• printer-friendly color should be used so that anyone can print it with a home printer
• black keys should be darker
• a black and white version would be appreciated as some users were colorblind


All these suggestions were taken into account to make the final product, as well as other suggestions like making versions with minor scales only and major scales only and versions with solfege notation.
